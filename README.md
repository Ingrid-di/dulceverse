# DulceVerse

DulceVerse ist ein kleiner Buchclub-Webauftritt, erstellt mit HTML und CSS. Ziel ist es, eine schöne, übersichtliche Webseite zu gestalten, die den Austausch über Bücher anregt.

## Technologien
- HTML5
- CSS3

## Ziel
Eine einfache und ansprechende Webseite für Leser*innen und Buchliebhaber*innen.
